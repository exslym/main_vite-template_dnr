# GOOGLE ANALYTICS account parameters:

//

## NoBounce (вариант для многостраничного сайта: MainPage, Page1, ...)

Название параметра:

```
NoBounce_MainPage
NoBounce_Page1
```

Описание:

```
Время на странице: Главная страница
Время на странице: Page1
```

Свойство или параметр:

```
NoBounce_MainPage
NoBounce_Page1
```
